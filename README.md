# react-ts-3d

* echarts-gl
* three-js
* ...

```shell
npm install

npm run dev
```

引入 @types/node

```shell
npm install -D @types/node
```

## echarts-3d

引入依赖

```shell
npm install echarts
npm install echarts-gl
```

导入

```ts
import * as echarts from 'echarts';
import 'echarts-gl';
```




## three-js

引入依赖

```shell
npm install stats-js # 显示帧率
npm install three@latest
npm install -D @types/three

npm install three-fbx-loader # 用于加载 fbx 模型
```
